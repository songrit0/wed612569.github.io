# GitHub Pages Website

เว็บไซต์นี้สร้างขึ้นด้วย GitHub Pages

## โครงสร้างไฟล์

```
.
├── index.html      # หน้าหลักของเว็บไซต์
├── styles.css      # ไฟล์ CSS สำหรับจัดรูปแบบ
├── script.js       # ไฟล์ JavaScript สำหรับฟังก์ชันต่างๆ
└── README.md       # ไฟล์อธิบายโปรเจค
```

## วิธีการใช้งาน

1. Fork หรือ Clone repository นี้
2. ไปที่ Settings ของ repository
3. เลือก Pages ในเมนูด้านซ้าย
4. เลือก branch `main` และ folder `/root`
5. คลิก Save
6. เว็บไซต์จะพร้อมใช้งานที่ `https://[username].github.io/[repository-name]`

## การปรับแต่ง

- แก้ไข `index.html` เพื่อเปลี่ยนเนื้อหา
- แก้ไข `styles.css` เพื่อเปลี่ยนรูปแบบ
- แก้ไข `script.js` เพื่อเพิ่มฟังก์ชันการทำงาน
